隐藏属性

Symbol 允许我们创建对象的“隐藏”属性，代码的任何其他部分都不能偶尔访问或重写这些属性。因为用Symbol作为键，在访问时必须用该Symbol去访问。

如果想在字面量中使用Symbol则需要[]

for in循环会跳过Symbol，Object.assign则不会。

全局Symbol

每一个新建的Symbol都是不同的，所以我们如果想要相同名字的Symbol一致就必须用到全局Symbol注册表，在其中创建Symbol并且在之后可以通过它得到相同名字的Symbol。

创建和读取都用Symbol.for(key)

Symbol.keyFor

Symbol.for(key)的一个反向调用，根据一个 全局Symbol获取它的名称。

js内部存在许多系统Symbol

如：

  • Symbol.hasInstance
  • Symbol.isConcatSpreadable
  • Symbol.iterator
  • Symbol.toPrimitive

等

垃圾回收

js的内存管理是自动的，所有我们创建的原始值，对象，函数等都是占用内存的。

js的内存管理概念为可达性，与Java的垃圾回收在一定程度上具有相似性。

将一些值作为根，从根出发，不可以通过引用到达的值将会被垃圾回收，在js引擎中有一个垃圾回收器在后台自动执行，监控所有对象的状态，并删除不可达对象。
